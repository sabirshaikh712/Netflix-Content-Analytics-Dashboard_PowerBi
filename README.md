# 🎬 Netflix Content Analytics Dashboard

## 📌 Overview

The Netflix Content Analytics Dashboard is an interactive Power BI project designed to analyze Netflix's content library and provide meaningful insights through visual storytelling.

The dashboard helps users explore content distribution, ratings, countries, movies, and TV shows using dynamic visualizations and interactive filters.

---

## 🚀 Features

* 📊 Total Titles Analysis
* 🎥 Movies vs TV Shows Distribution
* 🌍 Top Countries by Content
* ⭐ Content Ratings Distribution
* 🔍 Interactive Filters

  * Type
  * Country
  * Rating
  * Release Year
* 📈 KPI Cards for Key Metrics

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization
* Interactive Slicers
* Business Intelligence Techniques

## 📊 Key Insights

* Netflix contains approximately **7.8K+ titles**.
* Movies contribute around **69%** of the content library.
* TV Shows contribute around **31%** of the content library.
* The United States has the highest number of titles.
* TV-MA is the most common content rating.
